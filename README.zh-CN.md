# VidiBot — 多平台视频自动化上传工具

> 一个通过授权制使用的闭源视频批量上传工具，  
> 支持多个国内外主流视频平台，无需调用官方 API。

---

## 📌 关于 VidiBot

VidiBot 是一款用于 **自动化批量上传视频内容** 的桌面工具，
通过浏览器自动化技术模拟真实用户操作，
帮助个人创作者、运营团队及企业高效完成多平台视频发布任务。

本项目为 **闭源商业软件**，通过授权方式提供完整功能使用权限。

---

## 🧠 为什么选择 VidiBot

- 🚀 批量自动上传，节省大量人工操作时间  
- 🌍 支持多个国内外主流视频平台  
- 🔐 授权制闭源软件，可稳定长期使用  
- 🤖 基于浏览器自动化，无需平台官方 API  
- 📊 支持多账号、多任务并行处理  

---

## 📦 功能亮点

- 多平台视频批量上传  
- 支持定时发布任务  
- 支持标题 / 描述 / 标签 / 封面设置  
- 支持上传结果日志与导出  
- 支持 Windows 系统稳定运行

## 🚀 快速上手

1. 下载并解压 VidiBot 程序  
2. 配置本地视频文件目录  
3. 添加需要上传的平台账号  
4. （可选）设置定时发布时间  
5. 点击“开始上传”即可运行
