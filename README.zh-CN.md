# VidiBot — 多平台视频自动化上传与批量发布工具

## 📌 关于 VidiBot

VidiBot 是一款商业授权的，闭源的，用于 **自动化批量上传视频内容** 的桌面工具，
通过浏览器自动化技术模拟真实用户操作，支持大部分视频发布平台，无需调用官方平台 API
帮助个人创作者、运营团队及企业高效完成多平台视频发布任务。

本项目为 **闭源商业软件**，通过授权方式提供无限制的视频上传数量。
> 本 GitHub 仓库仅用于产品介绍与说明，
> 不包含 VidiBot 的源代码或可运行核心逻辑。
---

## 🧠 为什么选择 VidiBot

- 🚀 批量自动上传，节省大量人工操作时间  
- 🌍 支持多个国内外主流视频平台  
- 🔐 授权制闭源软件，可稳定长期使用  
- 🤖 模拟真实人工操作，更接近平台原生使用方式  
- 📊 支持多账号、多任务并行处理  

---

## 📦 功能亮点

- 多平台视频批量上传  
- 支持定时发布任务  
- 支持标题 / 描述 / 标签 / 封面设置  
- 支持上传结果日志与导出  
- 支持 Windows 系统稳定运行

## 🚀 快速上手

1. 下载并解压 VidiBot 程序包，无需任何编程基础，解压即可使用。
3. 配置指纹浏览器，新建指纹窗口，用这个窗口登录到视频平台后台
4. 把视频文件、缩略图、视频说明等放在同一个文件目录  
5. 打开VidiBot，新建一个上传任务，指定文件目录，并为之指定一个指纹窗口
6. （可选）设置定时发布时间  
7. 点击“开始”即可运行
