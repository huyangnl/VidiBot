# 授权说明 & 常见问题（FAQ）

本文档用于说明 VidiBot 的授权方式、使用边界及常见问题，  
帮助你判断 **是否适合使用 VidiBot，以及应选择哪种授权方式**。

---

## 一、VidiBot 是开源软件吗？

**不是。**

VidiBot 是一款 **闭源商业软件（Proprietary Software）**。  
GitHub 仓库仅用于产品介绍、版本发布与文档说明，**不提供源代码**。

---

## 二、可以免费使用吗？

**可以。**

在未填写授权码的情况下，VidiBot 将以 **免费模式（Free Mode）** 运行：

- 功能完整
- 存在使用次数 / 规模限制
- 不承诺技术支持与稳定性 SLA
- 适合测试、体验与轻度使用

免费模式的目的是：  
👉 **让你在付费前，确认它是否真的适合你的业务场景。**

---

## 三、什么情况下需要购买授权？

当你符合以下任意情况时，**建议购买授权**：

- 需要 **更高的发布频率或任务规模**
- 同时运营 **多个账号或多个平台**
- 用于 **商业用途 / 团队 / 工作室**
- 希望获得 **更稳定的版本更新与问题修复**
- 希望长期、持续地使用 VidiBot

---

## 四、授权后我能获得什么？

在激活授权后，你将获得：

- 解锁免费模式下的使用限制
- 支持更高强度的自动化任务
- 可用于个人、团队或商业场景（以授权类型为准）
- 持续的软件更新与功能迭代
- 更优先的问题修复与支持响应

> 不同授权类型在使用范围与设备数量上可能不同，  
> 以实际购买时的授权说明为准。

## 五、授权是按什么收费的？

VidiBot 的授权 **是按月或季或年订阅制**：

- 以 **授权码形式** 提供
- 授权策略可能随版本调整

如需具体授权方案，请直接联系作者确认。

## 六、授权是否绑定设备？

通常情况下：

- 授权码会与 **设备环境 / 机器码** 绑定
- 更换设备或系统可能需要重新授权
- 合理使用情况下可沟通调整

👉 这是为了防止授权被滥用，而不是限制正常用户。

## 七、可以把 VidiBot 转售、出租或对外提供服务吗？

**不可以。**

无论是否购买授权，以下行为都是 **明确禁止的**：

- 转售、出租、共享授权
- 将 VidiBot 作为产品或服务对外提供
- 以 SaaS、代运营工具、内部系统等形式二次封装
- 提供“代发布 / 批量发布”服务给第三方

如有 **定制或商业合作需求**，请提前沟通。

## 八、使用 VidiBot 会不会封号？

这是一个很常见、也很现实的问题。

需要明确的是：

- VidiBot **不保证账号绝对安全**
- 任何自动化工具都存在一定风险
- 是否封号取决于：
  - 平台规则
  - 账号质量
  - 使用频率
  - 操作行为是否异常

VidiBot 通过 **真实浏览器自动化** 尽量模拟人工操作，  
但 **最终风险仍由使用者自行承担**。

## 九、如果平台页面改版，软件还能用吗？

- 视频平台页面经常更新，这是事实
- VidiBot 会 **持续适配主流平台的 UI 变化**
- 更新会通过新版本发布进行修复

👉 这也是授权用户能获得持续价值的重要原因之一。

## 十一、我适合使用 VidiBot 吗？（快速判断）

你 **非常适合**，如果你：

- 经常需要重复上传相同或相似内容
- 同时运营多个视频平台
- 不想把时间浪费在机械操作上

- 只偶尔上传一两个视频
- 完全不接受任何自动化风险
- 期望“绝对不封号”的保证
